Project information

  • Category: CMS / ERP - Rest API - APP
  • Client: Sendit S.A
  • Project date: Ene 2023 - Dic 2023

Description

It is a company in charge of courier, it has several systems and an application for clients.
The first system is an ERP that allows billing, membership module, discount module, automatic debits, among other modules.
The following CMS system allows the control of the content displayed in the application, as well as the control of notifications to users, news control, among others.
Finally, the application allows you to verify the information of your package, as well as the current status of the shipment, shipping price, package price, and to be able to pay through the application with banking entities.
